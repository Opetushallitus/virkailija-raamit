# virkailija-raamit
Virkailijan työpöydän navigaatiovalikko.

### Raamien lokaali kehitys
`cd src/main/static/`

`npm install`

`npm start`

Mene selaimella: `http://localhost:3000`

### Prod buildin kehitys
`mvn clean install`

Uudelleennimeä target kansion .war => `virkailija-raamit.war` ja aja tomcatilla
